## 安装

This extension for Visual Studio Code adds themes titled `Qtz HIS AI`.

1. Open **Extensions** sidebar panel in Visual Studio Code. `View → Extensions`
2. Search for `Qtz HIS AI`
3. Click **Install**
4. Click **Reload**
